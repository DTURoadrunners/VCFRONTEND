About

This project is created by five software students at DTU (Technical University of Denmark) in association with DTU Roadrunners and DTU Mathematics and Computer Science. The project is designed to construct a version management system where guests, engineers and teachers can view, save and edit data and documentation for multiple projects. It must be subsequently documented for future projects, such as chemistry and other institutions at DTU.
